## Run Locally

Clone the project

```bash
  git clone https://github.com/ilhamnn/Fundamental-Front-End-Web-Development-2.git
```

Go to the project directory

```bash
  cd project folder
```

Install dependencies

```bash
  npm Install
```

```bash
  npm run build or npm run start-dev
```

Start the server
